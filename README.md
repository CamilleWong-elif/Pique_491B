# Pique_491B

## Running the App

### Everyone (except Camille)

```bash
cd pique_491b
npm install  # only needed the first time
npx expo start --tunnel
```

### Camille (Mac)

```bash
cd pique_491b
npm install  # only needed the first time
chmod +x node_modules/.bin/expo
npx expo start --tunnel
```
