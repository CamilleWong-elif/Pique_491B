# Pique Event Booking/Browsing Application
## Getting started

### Set up Expo Go environment
1. Install dependencies, use within IDE (VS Code) if preferred:

   ```bash
   npm install
   npm install -g expo-cli lucide-react-native react-native-svg 
   npx expo install expo-splash-screen react-native-reanimated 
   ```

2. Start the app

   ```bash
   npx expo start
   ```
   On school or public wifi, use this command if having connection issues:
   ```bash
   npx expo start --tunnel
   ```

3. For API key, I left a placeholder .env.example file to replace. Generate an API key and add to .env file (make sure to add to .gitignore as well).
